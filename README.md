WeShoot
=======

微信网页弹幕chrome插件

chrome商店链接：
https://chrome.google.com/webstore/detail/weshoot/dacmbpedcnehojlcbcihbhkdhkeaonio?utm_source=chrome-ntp-icon

安装后登陆网页版微信或网页版QQ，选择要捕获消息的群或好友，然后就可以在其他标签下看到弹幕了。
